# 图书管理系统

## 功能概述
这是一个基于Flask的图书管理系统，提供了完整的图书管理功能，包括：
- 图书信息管理（增删改查）
- 图书搜索功能
- 库存管理
- 图书分类管理

## 技术栈
- Flask (Web框架)
- SQLAlchemy (数据库ORM)
- Bootstrap 5 (前端样式)
- jQuery (前端交互)

## 安装和运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行系统
```bash
python run_book_system.py
```

### 3. 访问系统
打开浏览器访问 http://localhost:5000

## 功能特性

### 图书管理
- **添加图书**: 在图书列表页面点击“添加图书”按钮，填写图书信息
- **编辑图书**: 在图书列表中点击“编辑”按钮，修改图书信息
- **删除图书**: 在图书列表中点击“删除”按钮，确认后删除图书
- **搜索图书**: 使用顶部搜索框按标题或作者搜索图书

### 数据库模型
系统使用SQLite数据库存储图书信息，主要字段包括：
- ID: 图书唯一标识
- 标题: 图书标题
- 作者: 图书作者
- ISBN: 国际标准书号
- 出版日期: 出版时间
- 出版社: 出版机构
- 分类: 图书分类
- 页数: 图书页数
- 价格: 图书价格
- 库存数量: 当前库存
- 描述: 图书简介
